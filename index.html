Research

I currently am employing Machine Learning and Natural Language Processing techniques in order to develop features for Command-Line Interfaces (CLIs) in order to make them easier for users to learn and become comfortable with. These features include autocorrect, predictive text, and a system for recommending relevant commands to users based on their own unique questions. My work is being advised by Dr. Kinga Dobolyi.
